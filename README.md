# RE-replace-lists-for-TextPro
[TextPro](http://www.fodianbaoku.com/tools/index.htm)风格的自定义替换表、批量正则替换表，可用于TextPro和[AI Proofreader](https://github.com/Fusyong/ai-proofread-vscode-extension)。文件名中的`RE`表示条目是正则表达式；`EX`表示扩展语法（条目中有注释），仅适用于AI Proofreader，删除注释后可用于TextPro。

本人比较常用的都存在这里，不一定对他人有用

欢迎提意见，尤其是指出错误

交流文本处理经验，请访问：

[blog.xiiiGame.com](https://blog.xiiigame.com/)
